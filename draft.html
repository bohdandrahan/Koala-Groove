This is a project where I hope to implement simple looper pedal.
I was hoping to find some app online for my guitar practice and I was surprised not to find what I was looking for, not even behind a paywall. So I desiced to write my own tool.
I'm not professional musician and never saw a real looper pedal or any professional set up for this purposein my life so I'm not even sure what exactly the standart functionality there is. But on the other hand I have a clear vision for what are my personal needs are, with maybe few bells and whistels down the line. There are two approaches I see for this tool:

  • Timer triggered loop
  • Loop by push of a button

Time Trigerred Loop


This is probably more functional loop tool, but it will require way more sophisticated knowlege in music theory.
It would probaly have to be tied to metronome, there should be some sort of set up for time signature. The way I see should look something like this:
  • I set up time signature, let's say 4/4 by default
  • I set up BPM, let's say 90
  • I set up the number of measures(or bars). Let's say 5
  • Maybe I need to set up timer before start, but this is optional, I can see 5 sec being default
  • I click "Start rectording"
  • I see a 5 second countdown, perhaps nicely visualised by a circle
  • When countdown stops, the recording begins
  • There should be visual cues where we are in the bar
  • Visual metronome sould be there as well tied to the bars
  • whenever the times ends, there should appear "Play/Pause" button and "Delete" button
  • Play would start the loop
  • Cool feature would be to be able to select a segment to play, let's say play just one selected bar, or selected custom range
